# XButton

XButton

### Usage

```ts
import { XButton } from 'xbc-design';

<XButton>按钮文案 <XButton>;
```

| 参数     | 说明                         | 类型    | 可选值 | 默认值 |
| -------- | ---------------------------- | ------- | ------ | ------ |
| type     | 按钮的类型                   | string  | 是     | ''     |
| size     | 按钮的大小                   | String  | 是     | normal |
| loading  | 设置按钮的载入状态           | Boolean | 是     | false  |
| disabled | 设置按钮为禁用态（不可点击） | Boolean | 是     | false  |
